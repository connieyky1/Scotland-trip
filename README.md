# 🏰 Edinburgh Travel App

Your personal iPhone-installable Edinburgh travel guide for 3–6 April.

## Files included
- `index.html` — the main app
- `manifest.json` — makes it installable on iPhone
- `sw.js` — lets it work offline
- `icon-192.png` & `icon-512.png` — app icons

---

## 🚀 How to put it on GitHub Pages (step by step)

### Step 1: Create a GitHub account
Go to [github.com](https://github.com) and sign up (free).

### Step 2: Create a new repository
1. Click the **+** button → **New repository**
2. Name it: `edinburgh-trip` (or anything you like)
3. Set it to **Public**
4. Click **Create repository**

### Step 3: Upload your files
1. On your new repo page, click **uploading an existing file**
2. Drag and drop ALL 5 files:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
3. Scroll down and click **Commit changes**

### Step 4: Enable GitHub Pages
1. Click **Settings** (top of your repo)
2. Scroll down to **Pages** in the left sidebar
3. Under **Source**, select **Deploy from a branch**
4. Choose **main** branch, **/ (root)** folder
5. Click **Save**

### Step 5: Get your link
- Wait 1–2 minutes, then your app will be live at:
  `https://YOUR-USERNAME.github.io/edinburgh-trip/`

---

## 📱 Install on iPhone (Safari)

1. Open the link in **Safari** on your iPhone
2. Tap the **Share** button (box with arrow pointing up)
3. Scroll down and tap **"Add to Home Screen"**
4. Tap **Add**
5. The Edinburgh app icon appears on your home screen! 🎉

> **Note:** Must use Safari (not Chrome) for iPhone installation.
